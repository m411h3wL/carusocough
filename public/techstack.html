Tech Stack

Node.js

Runtime environment. Runs the Express server.

Express

Web framework for Node.js. Serves static files and handles the recording upload API.

Multer

Middleware for handling multipart/form-data. Used to save audio recordings to disk.

MediaRecorder API

Browser API for recording audio. Captures microphone input and saves as .webm files.

SoulX-Singer

Zero-shot singing voice synthesis and conversion model. Supports melody (F0) and score (MIDI) conditioning.
