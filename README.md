# Caruso Cough — Vocal Trainer

A web-based vocal training app inspired by Enrico Caruso. Built with Node.js and Express.

## Features

- Vocal exercises with guided technique
- Anatomy references (air pathways, diaphragm, pharynx)
- Audio recording and playback
- Legendary singers section

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Stack

- **Backend:** Node.js, Express
- **Audio:** Browser MediaRecorder API, stored as `.webm`
- **Frontend:** Vanilla HTML/CSS/JS
